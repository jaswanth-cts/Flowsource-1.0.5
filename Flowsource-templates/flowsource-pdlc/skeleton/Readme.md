# Flowsource PDLC Skeleton

## Overview
This folder (`flowsource-pdlc`) contains the PDLC plugin skeleton, which is essential for enabling PDLC features in your Flowsource Core project.

## How It Works
- During packaging, the contents of the `plugins` folder from `flowsource-core` are copied here.
- This ensures you always have the latest version of the PDLC plugin template available.

## Usage Instructions
1. **Copy the Skeleton:**
   - When you need to use the PDLC plugin, copy the `flowsource-pdlc` folder from this location to the `plugins` section of your `flowsource-core` project.
2. **Integrate with Flowsource Core:**
   - After copying, Flowsource Core will recognize and utilize the PDLC plugin for your workflows.

## Notes
- Make sure to keep this skeleton updated to benefit from new features and improvements.
