# Microfrontend Shell App with React.js Template

This template provides a basic microfrontend shell application using React.js. The code is hosted in GitHub.

## Description

This React.js template provides a starting point for building microfrontend shell applications. You can modify and extend the template according to your requirements.

## Contents / Final Output

- A basic microfrontend shell application using React.js.

## Prerequisites

Before using this template, make sure you have the following:

- GitHub account.

## Getting Started

To get started with this template, follow these steps:

1. Clone the repository from GitHub to your local machine.
2. Open the project in your preferred IDE or text editor.
3. Modify the code according to your requirements.
4. Commit and push the changes to the GitHub repository.

## Installation
1. Install the dependencies using below command:
    
   yarn install
   
## Development
2. Start the development server using below command:

   yarn start
